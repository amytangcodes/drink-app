## How To Use

To clone and run this application, in your command line:

```bash
# Getting Started
$ git clone git@github.com:amytangcodes/drink-app.git
$ cd drink-app
$ bundle
$ cd client
$ npm install
$ cd ..
$ rails db:migrate
$ rails db:seed

# Run the app
$ rake start
```
